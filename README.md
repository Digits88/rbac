### How do I get set up? ###

* Install database
* Copy all *.copy.php into general files *.php
      - config/config.copy.php
      - config/database.copy.php
      - config/ion_auth.copy.php
      - libraries/Apps.copy.php
* Customize your setting
      - config/config.php
      - config/database.php
      - config/ion_auth.php
      - libraries/Apps.php
* Try your application
